Multiwallet setup

We've detected multiple wallet.dat files in your Particl folder. Particl Wallet can interact with multiple wallet files at once. Here you can select all the wallets you would like to use in this session:

{{ wallet }}
  • This dialog will open every time when multiple wallets are present (your wallet selection isn't permanent).
  • If you're unsure which wallet to select, choose all of them.
